# Angioedema

Non-pitting subcutaneous/submucosal swelling[^28405953]

## Mechanisms

- Histamine-mediated
  - Mast cell degranulation following exposure to allergen
  - Type 1 IgE mediated hypersensitivity reaction
- Bradykinin-mediated
